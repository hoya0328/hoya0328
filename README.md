## About Me

컴퓨터공학을 전공하며 **제품 기획과 기술 구현을 함께 이해하는 PM**으로 성장하고 있습니다.

사용자의 문제를 구체적인 제품 경험으로 바꾸고, 핵심 기능을 빠르게 구조화해 실제로 작동하는 프로토타입까지 만드는 일을 좋아합니다. 서비스 기획, UX 흐름, 비즈니스 모델과 기술 구현을 하나의 제품 과정으로 연결하는 것이 강점입니다.

AI 영상 스타트업에서 약 **40개월간 System Engineer**로 근무한 경험을 통해 제품이 기획과 개발에서 끝나지 않고 실제 환경에서 지속적으로 제공되는 과정까지 바라보게 되었습니다. 이 경험은 구현 가능성과 운영 현실을 함께 고려하는 제품 의사결정의 기반이 되었습니다.
