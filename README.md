## About Me

AI 영상 스타트업에서 약 **40개월간 System Engineer**로 근무하며 실제 서비스 환경을 경험했습니다.  
그 과정에서 좋은 기능만큼이나 서비스가 안정적으로 제공되고 지속적으로 운영되는 구조가 중요하다는 것을 배웠습니다.

현재는 컴퓨터공학과 3학년으로, 다양한 서비스 기획·개발 경험을 시스템과 인프라에 대한 관심으로 연결하고 있습니다. Linux와 네트워크를 기반으로 AWS, Docker, Terraform, CI/CD, Monitoring을 학습하며 **Cloud Engineer**를 준비하고 있습니다.

장기적으로는 DevOps · SRE · Platform Engineering까지 전문성을 넓혀, 높은 안정성과 신뢰성이 요구되는 **금융 IT와 금융 클라우드** 환경에서 기여하는 엔지니어를 목표로 합니다.
